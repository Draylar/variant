### Variant

---

![](assets/creeper_banner.png)

Introduces fun visual & mechanical entity modifiers to spice up vanilla entities.

Available modifiers ([visit the wiki page for descriptions](https://github.com/Draylar/variant/wiki)):
- Ghostly
- Thundering
- Shiny
- Flame
- Mega
